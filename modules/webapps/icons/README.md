# Webapp Icons

This directory contains icons for web app launchers.

## Required Icons

Place PNG icons (preferably 144x144 or larger) in this directory:

- `youtube.png` - YouTube
- `youtube-music.png` - YouTube Music
- `claude-ai.png` - Claude AI
- `chatgpt.png` - ChatGPT
- `github.png` - GitHub
- `teams.png` - Microsoft Teams
- `nixos.png` - NixOS
- `gmail.png` - Gmail
- `calendar.png` - Google Calendar
- `x.png` - X

## Usage

Icons are automatically copied to `~/.local/share/icons/webapps/` during webapp setup.
